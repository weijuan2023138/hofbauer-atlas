## 图5 — 疾病中HB通讯网络的重塑

### 科学问题

妊娠疾病（感染、PE、流产）是否破坏图4建立的正常HB通讯架构？如果破坏，是"全局崩塌"还是"通路特异性重编程"？不同疾病之间是否存在共同的通讯脆弱节点？

### 方法

**5a: 感染 vs 对照全细胞CellChat比较**

| 参数 | 内容 |
|------|------|
| 数据 | hoo_2024（158,978细胞，15种细胞类型） |
| 分组 | Control vs Toxoplasma vs Malaria vs Listeria |
| 鉴定 | 使用已有celltype_annotation，无需重新注释 |
| 分析 | 每组独立跑CellChat，比较：入/出方向L-R对数量、通讯概率top15通路变化、关键分子（SPP1/FN1/TGF-β）的通讯强度 |
| 输出 | 气泡图对比、差异L-R对火山图、通路热图 |

**5b: PE/流产中HB通讯基因表达差异**

| 参数 | 内容 |
|------|------|
| 数据 | unified_hofbauer.rds（13,582 HB，含PE/流产/正常标签） |
| 基因集 | 图4鉴定的核心通讯基因：SPP1、FN1、COL1A2、COL1A1、TGFB1、IGF1、PTPRM、CD44、ITGAV、ITGB1等（~30个） |
| 分析 | PE vs Normal、流产 vs Normal 差异表达；GSEA验证通讯通路是否整体改变 |
| 输出 | 火山图、热图 |

**5c: 跨疾病通讯脆弱节点**

整合5a（感染全CellChat）和5b（PE/流产通讯基因DEG），鉴定在多种疾病中共同改变的通讯分子。这些"脆弱节点"可能是妊娠疾病的共同治疗靶点。

### Panel 布局

```
a  感染CellChat全景对比（气泡图，Control vs Infected）
b  感染前后L-R对数量变化（柱状图，入/出方向）
c  PE/流产通讯基因火山图
d  跨疾病保守变化的热图（SPP1/FN1/TGF-β等在三种疾病中的变化方向）
```

### 预期发现

- SPP1-FN1-COLLAGEN 轴在感染中显著上调（HB→免疫细胞输出增强），在PE中可能下调（胎盘缺氧导致HB功能障碍）
- TGF-β 免疫抑制信号在感染和PE中均下降，提示疾病状态下HB的免疫稳态功能受损
- 存在2-3个跨疾病的共同通讯脆弱节点

## 图6 — TF编程驱动通讯基因表达

### 科学问题

图2鉴定的发育TF开关（CEBPA→STAT3/STAT1）是否直接调控图4发现的核心通讯基因？如果是，这个TF→通讯基因→细胞功能的调控轴是否构成了HB通讯身份转变的分子基础？

### 方法

| 步骤 | 方法 | 输入 |
|------|------|------|
| 6a: TF-通讯基因相关性 | 计算STAT1/3、CEBPA、NFKB与30个通讯基因在13582个HB中的Pearson相关性 | unified_hofbauer.rds表达矩阵 |
| 6b: ATAC-seq通讯基因座可视化 | 提取SPP1、FN1、COL1A2、TGFB1基因座±50kb的ATAC信号，对比中期vs足月 | Fig2 ATAC-seq数据 |
| 6c: Motif扫描 | FIMO扫描通讯基因启动子（−2kb至+500bp）和ATAC开放峰中的STAT3/CEBPA/NFKB结合基序 | JASPAR motif矩阵 + ATAC峰坐标 |
| 6d: 整合模型 | TF活性 → 通讯基因表达 → 靶细胞功能，构建从核内调控到细胞间对话的完整链条 | 6a/b/c结果汇总 |

### Panel 布局

```
a  TF-通讯基因Spearman热图（STAT1/3、CEBPA、NFKB × 30通讯基因）
b  SPP1/FN1基因座ATAC轨迹（中期 vs 足月）
c  Motif富集条形图（通讯基因启动子中的TF结合位点统计）
d  调控模型示意图（STAT3→SPP1→vEC调控轴）
```

### 预期发现

- STAT3与SPP1/FN1表达强正相关（r > 0.6），CEBPA与COL1A2/ITGA1表达正相关
- 足月HB在SPP1启动子区域获得新的STAT3结合峰
- 通讯基因启动子中STAT和NFKB motif密度显著高于随机基因组背景

## 图7 — HB亚型的通讯基因表达分工

### 科学问题

图1鉴定的6个HB亚型是否表达了不同的L-R分子库？如果是，各亚型在通讯网络中承担什么角色？这种分工是否与亚型的发育时序（早期→晚期）和空间定位（血管周→间质深部）一致？

### 方法

| 步骤 | 方法 |
|------|------|
| 7a: 亚型×通讯基因点图 | 6个亚型 × 30个通讯基因的dotplot，按功能分类着色 |
| 7b: 通路级评分 | 每个亚型的ECM通讯评分、TGF-β通讯评分、免疫通讯评分、生长因子通讯评分 |
| 7c: 亚型-通讯功能映射 | 将6个亚型映射到通讯网络的"发送/接收/双向/沉默"四种角色 |

### Panel 布局

```
a  Dotplot：6亚型 × 30通讯基因（列按功能分四组：ECM/TGF-β/生长因子/免疫）
b  Boxplot：四个通讯模块评分在6个亚型中的分布
c  功能映射图：每个亚型在通讯网络中的角色（发送者/接收者/双向/沉默）
d  （可选）亚型与Fig3空间定位的对应关系
```

### 预期发现

| 亚型 | 预测通讯角色 | 依据 |
|------|------------|------|
| 血管重塑 | SPP1/FN1主要分泌者 | 这两种基因是该亚型标志物 |
| MHCII+ 抗原呈递 | 免疫输出主力 | HLA+SPP1向dNK/CD14_M广播 |
| 稳态/Homeostatic | 接收ECM信号终端 | 高表达整合素受体ITGAV/ITGB1 |
| PRKN+自噬 | 代谢/应激信号感知 | 接收TGF-β/BMP信号 |
| 促炎性/Inflammatory | 双向活跃但偏向输出 | CXCL8+SPP1高表达 |
| 增殖/Proliferating | 通讯沉默（低表达所有L-R） | 细胞周期为主，暂不参与通讯 |

## 交互式数据探索网站

### 目标

将图1-7的全部数据整合为一个交互式Shiny应用，供读者自主探索Hofbauer细胞的发育图谱、通讯网络和疾病变化。

### 功能模块

| 模块 | 内容 | 交互方式 |
|------|------|---------|
| **发育图谱** | UMAP按亚型/孕期/疾病着色 | 下拉选择 |
| **基因表达** | 输入基因名，展示UMAP + 小提琴图（6亚型 × 3孕期） | 文本框搜索 |
| **通讯网络** | CellChat气泡图 + chord图，可切换入/出方向、选择细胞对 | 下拉+点击 |
| **疾病对比** | PE vs Normal、感染 vs Control 的火山图 + 通讯基因热图 | 下拉选择 |
| **TF调控** | TF-通讯基因相关性热图 + 基因座ATAC轨迹 | 点击基因名联动 |
| **数据下载** | 表达矩阵、DEG列表、CellChat结果 | 下载按钮 |

### 技术方案

```
框架：R Shiny + bslib主题
后端数据：预计算的RDS文件（统一表达矩阵 + CellChat对象 + DEG结果）
绘图：plotly（UMAP散点图、火山图）+ ggplot2（静态图）+ visNetwork（网络图）
部署：shinyapps.io 或 自建服务器
文件总大小：<500MB（压缩RDS）
```
