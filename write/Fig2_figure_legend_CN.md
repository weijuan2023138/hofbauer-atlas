# 图2. 转录因子编程和染色质重塑驱动Hofbauer细胞的发育转换。

（A）孕晚期与孕早期Hofbauer细胞之间47个显著差异表达转录因子（TF）的点图（|log2FC| > 0.5，FDR < 0.01）。22个TF在孕早期富集（包括CEBPA、MAFB、ID2、SOX4），25个TF在孕晚期富集（包括STAT1、STAT3、NFKB1、RELB、NR4A3）。STAT1和STAT3上调幅度最大（log2FC分别为2.84和2.17），CEBPA下调显著（log2FC = −1.93）。

（B）足月（GW32–38）与孕中期（GW11–24）FACS纯化Hofbauer细胞的ATAC-seq差异可及性峰火山图。红点，足月获得峰（373个，FDR < 0.05，|log2FC| > 1）；蓝点，孕中期获得峰（31个）。足月特异性开放峰的总信号强度约为中期特异性峰的12倍，表明染色质重塑呈不对称性，以调控元件获得为主。

（C）足月获得ATAC-seq峰的de novo motif富集分析。按E-value排序的前五个TF结合基序为：STAT3（E-value = 3.2 × 10⁻¹²，占足月峰的7.8%）、STAT1（5.7 × 10⁻¹⁰）、RELB（8.1 × 10⁻⁹）、NFKB2（2.3 × 10⁻⁸）和KLF4（1.6 × 10⁻⁷）。motif富集谱与（A）图中TF表达变化一致。

（D）ATAC-seq染色质可及性（x轴，峰信号log2 fold change）与RNA-seq基因表达（y轴，log2 fold change）的整合散点图，覆盖2,141个同时具有两种数据类型的基因。其中1,969个基因（91.9%）呈现方向一致的变化（Cohen's κ = 0.84，p < 2.2 × 10⁻¹⁶）。关键免疫效应基因（C1QA、FCGR3A）已标注。

（E）1,969个染色质-转录正向偶联基因的GSEA分析（Hallmark基因集）。下调程序聚焦于细胞周期（纺锤体组装、G2M检查点，FDR < 10⁻²⁰）和线粒体翻译（FDR < 10⁻¹⁵）。上调程序以适应性免疫为核心：抗原加工与呈递（MHC class I/II，FDR = 6.3 × 10⁻¹⁸）、补体激活（FDR = 2.1 × 10⁻¹⁴）和Fcγ受体介导的吞噬作用（FDR = 8.9 × 10⁻¹²）。

（F）45个代表性基因的热图，分为四个功能模块：发育（CEBPA、MAFB、SQSTM1）、代谢、组织重塑和免疫（C1QA/B/C、FCGR3A）。列按孕周排列的单个样本。右侧面板展示各孕期模块评分的箱线图：免疫模块增幅最大（Δ = 1.09标准单位），与发育和代谢模块呈最强负相关（Pearson r = −0.87）。

（G）总结面板，综合展示TF-染色质-转录调控轴：CEBPA/MAFB/ID2祖细胞TF程序被STAT1/STAT3/NFKB免疫TF程序替代；STAT3和STAT1作为先锋因子在免疫效应基因位点打开染色质；染色质重塑与转录激活广泛偶联（92%一致性）；功能转换遵循代谢→重塑→免疫的时序层级。
