# 图3. Stereo-seq揭示Hofbauer细胞在胎盘绒毛间质中的空间组织。

（A）16例中孕期（GW18+6至GW24+1）健康胎盘基底板的Stereo-seq空间转录组图谱（bin50，约50 μm直径）。每个样本左图：所有10种注释细胞类型的空间分布，按颜色标注；右图：Hofbauer细胞（HB，红色）与其他所有细胞（灰色）的对比。标注样本编号和孕周。HB细胞聚集于绒毛间质区域，极少出现于绒毛外滋养细胞壳（EVT shell）或蜕膜深层。

（B）代表性绒毛截面（直径约180 μm）的高倍放大（40×）观察，取自Sample 011（GW20+5）。HB细胞（红色）紧贴胎儿毛细血管分布，胞体与血管的直接距离低于5 μm。HB被成纤维细胞（FB，绿色）和胎儿血管内皮细胞（fVEC，蓝色）包围，形成"FB包裹HB、HB贴近血管"的三明治式空间排列。SCT/VCT边界由蓝色虚线标注，包围绒毛间质。

（C）Hofbauer细胞50 μm半径邻域内各细胞类型的邻域富集分析。柱状图展示每种细胞类型相对于10,000次空间随机置换的观测/期望比值（富集倍数）。FB富集最强（2.77倍，p < 10⁻⁴，Bonferroni校正），其次为fVEC/fEC（1.91倍，p < 10⁻⁴）。SCT（0.31倍）和VCT（0.44倍）显著贫化，与滋养层和间质核心之间80–120 μm的空间隔离一致。母体免疫细胞（dNK、CD14_M）在HB邻域中几乎不存在（富集倍数 < 0.1）。红色柱，富集（富集倍数 > 1.2）；蓝色柱，贫化（富集倍数 < 0.8）；灰色柱，不显著。

补充图3A. 所有16例Stereo-seq样本的细胞类型组成热图，HB列以红色突出显示。数值代表每种细胞类型在各样本中的百分比。HB比例在样本间约为3%–8%，确认其一致性。

补充图3B. Stereo-seq数据中典型Hofbauer细胞标志基因（FOLR2、CD163、CD68、C1QA、C1QB、C1QC、SPP1、FN1等）的点图，确认空间注释HB细胞的分子身份。

补充图3C. Sample 010（GW20+3）和011（GW20+5）的绒毛放大图，显示HB空间组织在不同样本间的一致性。FB、fVEC和HB在绒毛间质内共定位，SCT/VCT构成外层边界。

补充图3D. 全部16例样本的深色背景空间图谱，HB细胞以亮红色突出显示。这些图像强调了HB细胞在所有检测孕周中的非随机、间质局限分布模式。
